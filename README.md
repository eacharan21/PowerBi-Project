# 📊 Personal Finance Dashboard – Power BI

---

## 🚀 Overview

This project presents an **interactive Power BI dashboard** designed to analyze personal financial data and provide meaningful insights into spending behavior and financial planning.

The dashboard helps users understand:

* Income vs Expenses
* Savings trends
* Category-wise spending
* Financial patterns over time

---

## 🎯 Objectives

* Analyze personal financial data
* Track income and expenses
* Identify spending patterns
* Support better financial decision-making

---

## 📊 Key Features

### 🔹 Financial Overview

* Total Income
* Total Expenses
* Net Savings

### 🔹 Expense Analysis

* Category-wise expense distribution
* Monthly spending trends

### 🔹 Income Analysis

* Income sources breakdown
* Growth trends over time

### 🔹 Insights

* High spending categories
* Savings behavior patterns
* Budget improvement suggestions
